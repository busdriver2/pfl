3D Reconstruction

December 13, 2023 by Matthew Shin

Structured illumination and meshing

The goal of this project is to turn multiple images of an object (teapot) into a 3d model in blender.

I used 10 bit graycode, converting to standard binary then decimal to get the output value. The images are in pairs, one for the original and one for the inverse. We compare the two frames to see which is brighter. By using structured light and projecting it onto the object, we can get away with using only one camera and one projector. After decoding, we can reconstruct a 3D point cloud and mesh the object. We clean up the mesh by pruning some points using Bounding box pruning and triangle pruning. Lastly, we smooth the mesh using laplacian smoothing and perform poisson reconstruction in meshlab.

Mesh coloring and alignment

The next step was to color the mesh using the images of the teapot without any projected light. This was achieved by reshaping the arrays using numpy operations and write to a .ply file. The mesh alignment was done in blender manually.
